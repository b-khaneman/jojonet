JojoNet v1.1.0
==============
Repo: https://github.com/b-khaneman/jojonet

INSTALL
-------
curl -fsSL https://raw.githubusercontent.com/b-khaneman/jojonet/main/install.sh | sudo bash
sudo jojonet --version
# jojonet 1.1.0

RUN
---
# Foreign first, then Iran:

# Best latency (UDP):
sudo jojonet --wg PEER_IP

# High speed / filtered:
sudo jojonet --hy2 PEER_IP

# Classic GRE:
sudo jojonet PEER_IP

# Fallbacks:
sudo jojonet --tun-tcp PEER_IP
sudo jojonet --paqet PEER_IP
sudo jojonet --vxlan PEER_IP

UNINSTALL
---------
sudo jojonet-uninstall
